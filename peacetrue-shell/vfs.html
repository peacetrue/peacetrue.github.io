-

普通文件，文本、图片、可执行。

目录。

c

字符设备，当前读取，例如：流、键盘。

b

块设备，随机读取，例如：硬盘、内存。

p

pipeline，管道。

s

socket，套接字。

1. 手动挂载分区

# 创建一个被 0 填充的 100M 镜像
$ dd \(1)
    if=/dev/zero \(2)
    of=mydisk.img \(3)
    bs=1048576 \(4)
    count=100 (5)
100+0 records in
100+0 records out
104857600 bytes (105 MB) copied, 0.0345331 s, 3.0 GB/s

$ ls -lh mydisk.img
-rw-r--r--. 1 root root 100M Sep 10 08:53 mydisk.img

# 挂载到环回接口
$ losetup /dev/loop0 mydisk.img
$ ls -l /dev/loop0
brw-rw----. 1 root disk 7, 0 Sep 10 09:00 /dev/loop0

# 格式化磁盘
$ mke2fs mydisk.img

$ mkdir mydisk
# 挂载磁盘
$ mount -t ext2 /dev/loop0 mydisk

$ df -h
