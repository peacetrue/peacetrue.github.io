功能

新增

新增示例，检查编码唯一性。

查询

分页查询示例信息，编码、名称模糊匹配，创建时间范围匹配，可按创建时间、修改时间排序。

查看

查看示例详细信息。

修改

修改示例信息，包括编码、名称、创建者、创建时间、修改者、修改时间，检查编码唯一性。

删除

删除示例信息，可单个或批量删除。
