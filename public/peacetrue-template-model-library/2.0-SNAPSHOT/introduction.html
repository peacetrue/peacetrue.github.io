简介

模型类库模板是一个模板项目， 用于支持快速开发基于单模块的组件， 包含基本的增删改查功能、接口和页面。 开发出来的组件可以在具体项目中直接使用。 例如：字典组件，即是基于此模板快速开发的。

模型类库模板使用一个【示例】模块作为样本，示例模块的属性包括：

  • 主键

  • 编码：唯一

  • 名称

  • 创建者

  • 创建时间

  • 修改者

  • 修改时间
