入门

本文用于指导读者通过命令行使用 目录模板引擎。

1. 下载

下载 入门示例，得到 quick-start.zip。

2. 渲染

执行以下命令：

# 解压 quick-start.zip
$ unzip quick-start.zip
$ cd quick-start

# 查看文件列表
$ ls -l

lrwxr-xr-x  1 xiayx  staff   9 Aug 10 17:39 antora
lrwxr-xr-x  1 xiayx  staff  28 Aug 10 17:40 antora-options.properties
lrwxr-xr-x  1 xiayx  staff  23 Aug 13 13:53 antora-variable.adoc
lrwxr-xr-x  1 xiayx  staff  30 Aug 10 17:40 antora-variables.properties
lrwxr-xr-x  1 xiayx  staff  29 Aug 12 11:10 default-options.properties
lrwxr-xr-x  1 xiayx  staff  22 Aug 10 17:39 template-engine.jar

# 启动命令行
$ java -jar template-engine.jar

# 执行渲染命令
shell:> evaluate -t file:antora -o file:antora-options.properties -v file:antora-variables.properties -r learn

# 退出命令行
shell:> quit

点击查看 渲染命令参数详解。

3. 对比

3.1. 对比文件结构

对比目录模板文件结构和生成结果文件结构，观察两者间的差异：

# 查看目录模板文件结构
$ tree antora
antora
|-- LICENSE
|-- README.adoc.vm
`-- docs
    |-- README.graffle
    `-- antora
        |-- antora.yml.vm
        `-- modules
            `-- ROOT
                |-- assets
                |   |-- attachments
                |   `-- images
                |-- examples
                |-- nav.adoc
                `-- pages
                    |-- branch.adoc
                    `-- index.adoc.vm

9 directories, 7 files

# 查看生成结果文件结构
$ tree learn

learn
|-- LICENSE
|-- README.adoc
`-- docs
    |-- README.graffle
    `-- antora
        |-- antora.yml
        `-- modules
            `-- ROOT
                |-- assets
                |   |-- attachments
                |   `-- images
                |-- examples
                |-- nav.adoc
                `-- pages
                    |-- branch.adoc
                    `-- index.adoc

9 directories, 7 files

  1. README.adoc.vm 变成 README.adoc

  2. antora.yml.vm 变成 antora.yml

  3. index.adoc.vm 变成 index.adoc

  4. local.adoc 在结果中不存在

  5. 其他文件两者相同

3.2. 对比 README.adoc.vm

$ cat antora/README.adoc.vm

= 简介
:app-name: ${repository.directoryName}

${repository.description}

$ cat learn/README.adoc

= 简介
:app-name: learn

记录学习过程。

3.3. 对比 antora.yml.vm

$ cat antora/docs/antora/antora.yml.vm

name: ${repository.directoryName}
title: ${repository.title}
version: master
asciidoc:
  attributes:
    numbered: '@'
    app-name: ${repository.directoryName}
start_page: ROOT:index.adoc
nav:
  - modules/ROOT/nav.adoc

$ cat learn/docs/antora/antora.yml

name: learn
title: 学习
version: master
asciidoc:
  attributes:
    numbered: '@'
    app-name: learn
start_page: ROOT:index.adoc
nav:
  - modules/ROOT/nav.adoc

3.4. 对比 index.adoc.vm

$ cat antora/docs/antora/modules/ROOT/pages/index.adoc.vm

= 简介
:app-name: ${repository.directoryName}

${repository.description}

$ cat learn/docs/antora/antora.yml

= 简介
:app-name: learn

记录学习过程。

4. 变量

模板中占位符的值，都是从变量属性文件中读取的。

查看变量属性文件：

antora-variables.properties
$ cat antora-variables.properties
#\u53D8\u91CF\u5C5E\u6027\u6587\u4EF6
#Sat Aug 13 13:56:01 CST 2022
repository.title=学习
repository.directoryName=learn
repository.description=记录学习过程。
repository.name=Learn
repository.packageName=com.github.peacetrue.learn
