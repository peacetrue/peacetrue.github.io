入门

本文将指导读者通过命令行使用目录模板引擎。

1. 前提条件

目录模板引擎基于 Java 开发，使用时要求系统上具有 Java 运行环境，版本在 1.8 及以上。可使用如下命令检查：

java
$ java -version

java version "1.8.0_51"
Java(TM) SE Runtime Environment (build 1.8.0_51-b16)
Java HotSpot(TM) 64-Bit Server VM (build 25.51-b03, mixed mode)

2. 目录模板

读者可直接下载 Antora 目录模板，做学习之用。 关于 Antora 目录模板 的更多信息，可查看 Antora 目录模板简介； 如需自定义目录模板，可参考 目录模板制作教程。

3. 命令行渲染

下载 目录模板引擎类库 和 变量配置文件。 将下载的 template-engine.jar、antora.zip、variables.properties 放在同一目录下 [1]， 执行以下命令：

渲染
# 启动命令行
$ java -jar template-engine.jar
# 查看渲染命令
shell:> help evaluate
NAME
        evaluate - 渲染目录模板

SYNOPSYS
        evaluate [[-t] string]  [[-v] string]  [[-o] string]  [[-r] string]

OPTIONS
        -t or --templatePath  string
                目录模板位置。文件路径使用 file:/path/to/template，类路径使用 classpath:/path/to/template
                [Optional, default = file:template]

        -v or --variablesPath  string
                变量属性文件路径。路径规则同上
                [Optional, default = file:variables.properties]

        -o or --optionsPath  string
                选项属性文件路径
                [Optional, default = file:options.properties]

        -r or --resultPath  string
                结果路径
                [Optional, default = result]
# 执行渲染命令
shell:> evaluate -t file:antora.zip -v file:variables.properties -r test
# 退出命令行
shell:> quit
# 查看生成的结果目录结构
$ tree test
