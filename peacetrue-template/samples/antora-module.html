Antora 模块目录模板

Antora 模块目录模板用于在 Antora 资源库中创建一个模块。 Antora 资源库是基于 Antora 目录模板 创建的资源库。

1. 目录结构

Antora 模块目录模板 的目录结构如下：

.
├── ${module.directoryName}
│   ├── build.gradle.vm
│   ├── gradle
│   │   └── wrapper
│   │       ├── gradle-wrapper.jar
│   │       └── gradle-wrapper.properties
│   ├── gradle.properties.vm
│   ├── gradlew
│   ├── gradlew.bat
│   ├── settings.gradle.vm
│   └── src
│       └── test
│           ├── java
│           │   └── ${clazz.path(${repository.packageName})}
│           │       └── ${uc.lu(${module.name})}
│           └── resources
│               ├── logback.xml.vm
│               └── logback.xsd
└── docs
    └── antora
        └── modules
            └── ROOT
                ├── assets
                │   └── images
                │       └── ${module.directoryName}
                ├── examples
                │   └── ${module.directoryName}
                └── pages
                    └── ${module.directoryName}.adoc

以 .vm 结尾的文件为模板，此类文件会被渲染，其他文件直接拷贝。

3. 渲染

    /** Antora 模块目录模板 */
    @Test
    void antoraModule() throws IOException {
        DirectoryTemplateEngineImpl templateEngine = new DirectoryTemplateEngineImpl();
        templateEngine.setResourcesLoader(ConditionalResourcesLoader.DEFAULT);
        templateEngine.setTemplatePredicate(VelocityTemplateEngine.VELOCITY_TEMPLATE_PREDICATE);
        templateEngine.setResultPathResolver(VelocityTemplateEngine.VELOCITY_RESULT_PATH_RESOLVER);
        templateEngine.setTemplateEngine(new VelocityTemplateEngine(new VelocityEngine()));
        String projectAbsolutePath = SourcePathUtils.getProjectAbsolutePath();
        String templateDir = String.format("file:%s/../docs/antora/modules/ROOT/examples/templates/antora-module", projectAbsolutePath);
        String resultDir = String.format("%s/../docs/antora/modules/ROOT/examples/results/learn-redis", projectAbsolutePath);
        templateEngine.evaluate(
                templateDir,
                ImmutableMap.<String, Object>builder()
                        .put("repository", ImmutableMap.of("name", "Learn", "directoryName", "learn", "description", "记录学习过程", "packageName", "com.github.peacetrue.learn"))
                        .put("module", ImmutableMap.of("name", "Redis", "directoryName", "learn-redis", "description", "学习 Redis"))
                        .build(),
                Options.builder().evaluateTemplatePath(true).build(),//渲染模板路径
                resultDir
        );
        Assertions.assertTrue(Files.exists(Paths.get(resultDir)));
    }
