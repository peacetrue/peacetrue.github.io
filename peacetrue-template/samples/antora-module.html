Antora 模块目录模板

Antora 模块目录模板 用于在 Antora 资源库 中创建一个模块。 Antora 资源库 是基于 Antora 目录模板 创建的资源库。

1. 文件结构

Antora 模块目录模板 文件结构如下：

antora-module
|-- ${module.directoryName}
|   |-- build.gradle.vm
|   |-- gradle
|   |   `-- wrapper
|   |       |-- gradle-wrapper.jar
|   |       `-- gradle-wrapper.properties
|   |-- gradle.properties.vm
|   |-- gradlew
|   |-- gradlew.bat
|   |-- settings.gradle.vm
|   `-- src
|       |-- main
|       |   |-- java
|       |   |   `-- ${clazz.path(${repository.packageName})}
|       |   |       `-- ${uc.lu(${module.name})}
|       |   `-- resources
|       `-- test
|           |-- java
|           `-- resources
|               |-- logback.xml.vm
|               `-- logback.xsd
`-- docs
    `-- antora
        `-- modules
            `-- ROOT
                `-- pages
                    `-- ${module.directoryName}.adoc.vm

17 directories, 10 files

2. 变量说明

Antora 模块目录模板 中使用到的变量如下：

Table 1. 根对象
名称 描述 类型 备注

repository

资源库属性

资源库

module

模块属性

模块
Table 2. 模块
名称 描述 类型 备注

name

名称

字符串

约定为 大写驼峰格式，例如：Redis

title

名称

字符串

例如：学习 Redis

description

描述

字符串

例如：记录学习 Redis 的过程

directoryName

目录名称

字符串

约定为 小写中划线格式，例如：learn-redis

4. 生成结果

生成结果文件结构如下：

learn-redis
|-- docs
|   `-- antora
|       `-- modules
|           `-- ROOT
|               `-- pages
|                   `-- learn-redis.adoc
`-- learn-redis
    |-- build.gradle
    |-- gradle
    |   `-- wrapper
    |       |-- gradle-wrapper.jar
    |       `-- gradle-wrapper.properties
    |-- gradle.properties
    |-- gradlew
    |-- gradlew.bat
    |-- settings.gradle
    `-- src
        |-- main
        |   |-- java
        |   |   `-- com
        |   |       `-- github
        |   |           `-- peacetrue
        |   |               `-- learn
        |   |                   `-- redis
        |   `-- resources
        `-- test
            |-- java
            `-- resources
                |-- logback.xml
                `-- logback.xsd

20 directories, 10 files

文件名中变量为路径时，会生成多个目录。 例如：${clazz.path(${repository.packageName})}，生成结果文件结构为：

└── com
  └── github
      └── peacetrue
          └── learn
