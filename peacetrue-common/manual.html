1. Bean 遍历

1.1. 准备

准备如下 json 用户数据：

json
{
  "id" : null,
  "name" : "admin",
  "password" : "",
  "roles" : [ {
    "id" : 1,
    "name" : "admin",
    "tags" : [ "read", "write" ]
  }, null, {
    "id" : null,
    "name" : "user",
    "tags" : null
  }, {
    "id" : 3,
    "name" : null,
    "tags" : null
  } ],
  "employee" : {
    "id" : 1,
    "name" : "Jone",
    "tags" : [ "good", "better" ]
  }
}
